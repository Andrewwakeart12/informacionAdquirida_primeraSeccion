  3. Syntaxis

    in CSS you'll use a selector to select the element in the DOM (Document Object Model) so you can stylize one (id: #idInCSS) or various elements (class : .classInCSS)with the css (also you can select by the name of the element such a input,h1,etc). so a selection in CSS looks like this:

    1. ID:

      #oneElement{ property: value; }

    2. Class:

      .multipleElements{ property: value; }

    have in mind that you also can use somethiin like:

    Tag:

    input{ property: value; } so basically if the selector does'nt have a "#" or "." its just a regular tag so the propertys in there are going to affect only the input tags or the h1 tags, etc.

  4. Colors

    they come in four maybe five formats and they are:

    1. HEX (Hexadecimal)

      this is hexadecimal number that goes from #000000 to #FFFFFF and its defined in that way with number from 0 to 9 and then after 9 from A to F so you have 16 numbers including the 0

    2. RGBA (Red, Green, Blue,Alpha)

      its a scale that goes from 0 to 255 and use the colors Red Green and Blue fot the scale (the alpha its just the opacity)

    3. HSL (Hue Saturation Light)

      its the artist method to select color and in html and CSS takes the value in each one of the propertys from 0% to 100%

    4. Prefab

      its a gamma of color that CSS have preconfigured so you can use it just saying the color like "red","blue", etc

    there are more but this 4 are the main color forma ts
