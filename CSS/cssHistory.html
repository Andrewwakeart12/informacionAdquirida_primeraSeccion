    Css what is?

  1. CSS stands for Cascadin Style Sheets
  2. CSS describes how HTMl elements are to be displayed on screen, paper, or in other media
  3. CSS saves a lot of work. it can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files

what is CSS 3?

CSS3 is the lastest evolution of the Cascading Style language and aims at extending CSS2.1. It brings a lot of new features and additions, like rounded corners, shadows, gradients, transitions or animations, as well as new layouts like multi-columns,flexible box or grid layouts
