Forms

  1. form tag

    Form tag is a tag that contains fields, the fields came in different formats but their porpose is the same, a field contains information, information that the form can send to the server through the action parametter, the way its send it its called a method, the method can be Get (where the information are send it in the url) or Post(this protects the information making it not visible)

  2. input tag

    the input tag works inside a form tag, this define an input of information, for using it you need a type that cant be:

    1. text

    2. buttom

    3. checkbox

    4. radio

    5. number

    6. color

    7. month

    8. file

    9. image

    10. range

    and a few more, this fileds are very usefull for filter information or simply have and very clear idea for what the user will send

  3. textarea tag:

    its a tag that define a field for text, the text in it cant be very long, also we can block the size of the area if we wanted with rows and cols, alseo we can block the rows and cols value, so the user can't change it

  4. select tag

    A select tag its a tag that defines a field with selectionable items in form of a dropdown list
