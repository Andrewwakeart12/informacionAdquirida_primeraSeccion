1. script tag

    defines a script, a scrip its a piece of code that makes the content of the page interactive, throught this we can send alerts or show information in console like the following= "hello :D"

  2. noscrip tag

    a noscrip define an alternative content for users that do not support client-side script

  3. applet tag

    the applet tag defines embedded applet and its not supported in html5

  4. embed tag

    the embed tag defines an embedded applet and its also not supported in html5

  5. object tag

    object tag defines an embedded object and its not supported for most of the browsers

  6. param tag

    defines a param for an object tag, also not supported in html5
