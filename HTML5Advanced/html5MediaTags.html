Media Tags

Media tags are a way to add media files throught html5 wich makes easyer to include it, just need to define the media url reference and you can reproduce every file Compatible with html5 like the followings:

    1. Video
      1. MPEG (.mpg | .mpeg)
      2. AVI (.avi)
      3. WMV (.wmv)
      4. QuickTime(.mov)
      5. RealVideo(.rm | .ram)
      6. Flash(.swf | .flv)
      7. Ogg(.ogg)
      8. WebM(.webm)
      9. MPEG-4 or MP$(.mp4)
    2. Audio
      1. MID (.mid | .midi)
      2. RealAudio (.rm | .ram)
      3. WMA (.wma)
      4. AAC (.aac)
      5. WAV (.wav)
      6. Ogg (.ogg)
      7. MP3 (.mp3)
      8. MP4 (.mp4)

        9. Only MP3, WAV and Ogg are supporting by ordinary Html
